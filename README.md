# 🖼️ Image Watermarking App

A simple desktop app to add custom text watermarks to your images — built with Python and Tkinter.

## ✨ Features

- 📁 Upload any image
- ✏️ Custom watermark text
- 🎨 Choose text color (White or Black)
- 📍 Set watermark position (e.g. Bottom Right)
- 💾 Save watermarked image instantly

## 🚀 Getting Started

### Prerequisites
- Python 3.x
- Pillow

### Installation

```bash
# Clone the repository
git clone https://github.com/Kaemon/Day85.git
cd Day85

# Install dependencies
pip install pillow

# Run the app
python main.py
```
